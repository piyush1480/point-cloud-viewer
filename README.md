# Django 3D Point Cloud Viewer

A web application built in Django that uploads and visualizes 3D point cloud files (.ply) in the browser using Three.js.

## Features
- Upload .ply files
- 3D rotate, zoom, orbit
- ASCII point cloud rendering
- Bootstrap UI

## Run locally
```bash
git clone https://github.com/YOUR_USERNAME/point-cloud-viewer.git
cd point-cloud-viewer
pip install -r requirements.txt
python manage.py runserver
